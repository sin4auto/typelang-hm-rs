型システム基盤モジュール

  • Type や Constraint など、型推論で扱うコアデータ構造を定義する。
  • 置換操作・自由型変数計算・単一化といった基本演算を提供する。
  • 型の整形表示や単純なデフォルト化もこのモジュールで完結する。
